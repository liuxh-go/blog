# 排序算法简介

# 一、定义

对某一序列根据某关键字进行顺序排列的算法

# 二、相关名词

  • 稳定:若a=b,排序前a在b前面,排序后a也在b前面
  • 不稳定:若a=b,排序前a在b前面,排序后a可能在b后面
  • 内排序:所有的排序过程都在内存中进行
  • 外排序:由于数据量太大,需要将数据放到磁盘中,通过磁盘和内存的数据传输才能完成排序
  • 空间复杂度:排序过程中消耗的内存空间大小
  • 时间复杂度:排序过程中消耗的时间长短
